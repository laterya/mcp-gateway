# 分流标签

Skill 使用五个标准分流角色。本文件将这些角色映射到仓库 issue tracker 中实际使用的标签字符串。

| Skill 中的角色名          | Tracker 中的标签名      | 含义                       |
| ------------------------- | ----------------------- | -------------------------- |
| `needs-triage`            | `needs-triage`          | 维护者需要评估此 Issue     |
| `needs-info`              | `needs-info`            | 等待报告者补充信息         |
| `ready-for-agent`         | `ready-for-agent`       | 规格完整，可由 AFK Agent 接手 |
| `ready-for-human`         | `ready-for-human`       | 需要人工实现               |
| `wontfix`                 | `wontfix`               | 不予处理                   |

当 skill 提及某个角色（如"应用 AFK-ready 分流标签"）时，使用此表中对应的标签字符串。

如需修改右侧标签名以匹配实际使用的词汇，直接编辑此表即可。
